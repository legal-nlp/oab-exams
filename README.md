
## Definition

OAB Exams are the Brazilian version of the BAR exam (USA).

## Data

This repository has a corpus with raw text files from:

+ the official OAB Exams made by FGV PROJETOS; and,
+ simulation exams provided by FGV DIREITO RIO.
